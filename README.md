# HackLab Manager v2.2

Простой запуск:
1. Скачай hacklab_manager.zip
2. Распакуй: unzip hacklab_manager.zip
3. Запусти: chmod +x install.sh && ./install.sh
4. Используй: hl

Инструменты:
- network_info - информация о сети
- port_check - проверка портов
- web_scanner - сканирование веб-сайтов
- и еще 9 инструментов

Обучение: hl learn
